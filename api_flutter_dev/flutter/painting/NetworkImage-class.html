NetworkImage
description

NetworkImage class abstract

Fetches the given URL from the network, associating it with the given scale.

The image will be cached regardless of cache headers from the server.

When a network image is used on the Web platform, the getTargetSize parameter of the ImageDecoderCallback is only supported when the application is running with the CanvasKit renderer. When the application is using the HTML renderer, the web engine delegates image decoding of network images to the Web, which does not support custom decode sizes.

See also:

  • Image.network for a shorthand of an Image widget backed by NetworkImage.
  • The example at ImageProvider, which shows a custom variant of this class that applies different logic for fetching the image.
