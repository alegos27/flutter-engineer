PhysicalShape class

A widget representing a physical layer that clips its children to a path.

Physical layers cast shadows based on an elevation which is nominally in logical pixels, coming vertically out of the rendering surface.

PhysicalModel does the same but only supports shapes that can be expressed as rectangles with rounded corners.

This example shows how to use a PhysicalShape on a centered SizedBox to clip it to a rounded rectangle using a ShapeBorderClipper and give it an orange color along with a shadow.
link

To create a local project with this code sample, run:
flutter create --sample=widgets.PhysicalShape.1 mysample

See also:

  • ShapeBorderClipper, which converts a ShapeBorder to a CustomClipper, as needed by this widget.
