TestFlutterView class

A FlutterView that wraps another FlutterView and allows faking of some properties for testing purposes.

This class should not be instantiated manually, as it requires a backing FlutterView that must be produced from a PlatformDispatcher.

See also:

  • WidgetTester.view which allows for accessing the TestFlutterView for single view applications or widget testing.
  • WidgetTester.viewOf which allows for accessing the appropriate TestFlutterView in a given situation for multi-view applications.
  • TestPlatformDispatcher, which allows for faking of platform specific functionality.
Implemented types
