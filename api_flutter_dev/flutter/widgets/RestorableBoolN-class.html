RestorableBoolN class

A RestorableProperty that knows how to store and restore a bool that is nullable.

The current value of this property is stored in the restoration data. During state restoration the property is restored to the value it had when the restoration data it is getting restored from was collected.

If no restoration data is available, value is initialized to the defaultValue given in the constructor.

See also:

  • RestorableBool for the non-nullable version of this class.
