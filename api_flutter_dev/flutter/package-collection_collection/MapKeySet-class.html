MapKeySet

MapKeySet<E> class

An unmodifiable Set view of the keys of a Map.

The set delegates all operations to the underlying map.

A Map can only contain each key once, so its keys can always be viewed as a Set without any loss, even if the Map.keys getter only shows an Iterable view of the keys.

Note that lookup is not supported for this set.
