TestDefaultBinaryMessengerBinding mixin

Overrides the ServicesBinding's binary messenger logic to use TestDefaultBinaryMessenger.

Test bindings that are used by tests that mock message handlers for plugins should mix in this binding to enable the use of the TestDefaultBinaryMessenger APIs.

Superclass Constraints
  • BindingBase
  • ServicesBinding
Mixin Applications
  • TestWidgetsFlutterBinding
